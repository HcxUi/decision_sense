# DecisionSense

DecisionSense is a privacy-first AI-powered mobile application designed to help users analyze, understand, and improve their daily decision-making patterns. Built with Flutter and powered by on-device AI, the application enables users to log decisions, track emotional influences, identify cognitive biases, and receive personalized insights while keeping all sensitive data secure on the user's device.

## Features

- Privacy-first architecture with local data processing
- Offline AI-powered decision analysis
- Decision Score calculation and progress tracking
- Mood and context-based decision logging
- Personalized recommendations and insights
- Local storage using Hive database
- Hybrid AI support with optional remote API fallback
- Modern and responsive Flutter UI
- English and Hindi language support

## Tech Stack

- Flutter & Dart
- Riverpod State Management
- Hive Database
- llama_cpp_dart
- GGUF On-Device AI Models
- REST API Integration

## Project Goal

DecisionSense aims to bridge the gap between cognitive psychology and mobile technology by helping users make more informed decisions, reduce decision fatigue, and develop greater self-awareness while maintaining complete control over their personal data.

## Future Enhancements

- Advanced analytics and visualization dashboards
- Enhanced cognitive bias detection
- Expanded offline AI capabilities
- Smart habit and decision trend tracking
